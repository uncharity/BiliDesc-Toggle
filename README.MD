# BiliDescToggle 📝

> B站视频简介显示控制工具 | Video Description Toggle Tool for Bilibili

---

## 🚀 核心功能

### 📝 简介显示优化
- 默认完整显示视频简介内容
- 双击A键快速切换简介显示/隐藏
